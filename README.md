# 🌤️ Weather Observation Tracker

## Project Overview

The Weather Observation Tracker is a Streamlit application developed using Python. It allows users to record, store, search, and analyze local weather observations.

This project demonstrates the use of Python fundamentals including functions, file handling, pandas, and Streamlit.

---

## Features

- Add new weather observations
- Store observations in a CSV file
- View all saved observations
- Search observations by city
- Search observations by date
- Display weather statistics
- Visualize weather trends with charts
- Download observations as a CSV file

---

## Technologies Used

- Python
- Streamlit
- Pandas
- CSV File Storage

## Installation

Install the required libraries:

```bash
pip install streamlit pandas
```

---

## Run the Application

Open a terminal inside the project folder and run:

```bash
streamlit run weather_track.py
```
